Glossary
********

.. glossary::

   QR code
   Quick Response code
        A two dimensional barcode developed by Denso Wave.
        
   mode
        The encoding used to represent the data in a QR code.
        
   version
        A version is one of 40 different possible sizes a QR code comes in.
     
   error   
   error level
        QR codes can use one of four possible error correction values. They
        are refered to by the letters: L, M, Q, and H.
